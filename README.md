# glm_bc_generator
script to convert data downloaded from German Weather Service (DWD) directly into boundary conditions (input data) of GLM hydrodynamic model

# how to use:
put the hourly data directly downloaded from DWD into data_dwd/hourly directory (in .txt format) and run the python script glm_met_data_creator.py. The output files will be written into glm_out directory
